Steal Power - Spells Steal Power Source Mythic Adventures pg. 114 School necromancy; Level arcanist 5, cleric 5, mesmerist 4, oracle 5, psychic 5, sorcerer 5, warpriest 5, witch 5, wizard 5 Casting Casting Time 1 standard action Components V, S Effect Range close (25 ft. + 5 ft./2 levels) Target one mythic creature Duration instantaneous Saving Throw Will negates; Spell Resistance yes Description You sap the power from a mythic creature, using it to restore your own. The target loses 1d4 uses of mythic power. If you have the ability to expend mythic power, you regain one use for each use the target loses.
