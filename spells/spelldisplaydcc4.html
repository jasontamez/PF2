Spells Vanish Source Advanced Player's Guide pg. 253 School illusion (glamer); Level arcanist 1, bard 1, magus 1, mesmerist 1, occultist 1, psychic 1, redmantisassassin 1, skald 1, sorcerer 1, wizard 1 Casting Casting Time 1 standard action Components V, S Effect Range touch Target creature touched Duration 1 round/level (up to 5 rounds) (D) Saving Throw Will negates (harmless); Spell Resistance yes (harmless) Description This spell functions like invisibility, except the effect only lasts for 1 round per caster level (maximum of 5 rounds).
