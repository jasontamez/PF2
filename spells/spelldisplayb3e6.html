Liveoak - Spells Liveoak Source PRPG Core Rulebook pg. 305 School transmutation; Level druid 6, hunter 6, shaman 7 Casting Casting Time 10 minutes Components V, S Effect Range touch Target tree touched Duration 1 day/level (D) Saving Throw none; Spell Resistance no Description This spell turns an oak tree into a protector or guardian. The spell can only be cast on a single tree at a time; while liveoak is in effect, you can't cast it again on another tree. Liveoak must be cast on a healthy, Huge oak. A triggering phrase of up to one word per caster level is placed on the targeted oak. The liveoak spell triggers the tree into animating as a treant. If liveoak is dispelled, the tree takes root immediately wherever it happens to be. If released by you, the tree tries to return to its original location before taking root.
