Minor Phantom Object - Spells Minor Phantom Object Source Advanced Race Guide pg. 39 School illusion (shadow) [shadow]; Level arcanist 4, mesmerist 4, psychic 4, redmantisassassin 4, sorcerer 4, wizard 4 (gnome) Casting Casting Time 1 minute Components V, S Effect Range 0 ft. Effect illusory, unattended, nonmagical object of nonliving plant matter, up to 1 cu. ft./level Duration 10 minutes/level (D) Description This spell functions as the minor creation spell, except the object created is a semi-real illusion. Any creature that interacts with the object may make a Will save, with success causing the object to cease to exist. A gnome casting this spell may make a Spellcraft check in place of any Craft check required to make a complex item.
