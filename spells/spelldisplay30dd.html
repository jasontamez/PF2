Detect Fiendish Presence - Spells Detect Fiendish Presence Source Agents of Evil pg. 16 School divination; Level arcanist 0, bard 0, cleric 0, magus 0, oracle 0, skald 0, sorcerer 0, warpriest 0, witch 0, wizard 0 Casting Casting Time 1 standard action Components V, S Effect Range 60 ft. Area cone-shaped emanation Duration concentration, up to 10 minutes/level (D) Saving Throw none; Spell Resistance no Description This spell functions like detect evil, except that it detects only outsiders with the evil subtype, as well as the lingering effects caused by their gifts, presence, and spells. It can also detect clerics and paladins of fiendish deities, including Asmodeus, archdevils, daemonic harbingers, and demon lords.
