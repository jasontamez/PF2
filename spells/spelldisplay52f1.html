Spells Remove Curse Source PRPG Core Rulebook pg. 332 School abjuration; Level adept 3, alchemist 3, arcanist 4, bard 3, cleric 3, inquisitor 3, investigator 3, medium 3, mesmerist 3, oracle 3, paladin 3, shaman 3, skald 3, sorcerer 4, spiritualist 4, warpriest 3, witch 3, wizard 4 Casting Casting Time 1 standard action Components V, S Effect Range touch Target creature or object touched Duration instantaneous Saving Throw Will negates (harmless); Spell Resistance yes (harmless) Description Remove curse can remove all curses on an object or a creature. If the target is a creature, you must make a caster level check (1d20 + caster level) against the DC of each curse affecting the target. Success means that the curse is removed. Remove curse does not remove the curse from a cursed shield, weapon, or suit of armor, although a successful caster level check enables the creature afflicted with any such cursed item to remove and get rid of it. Remove curse counters and dispels bestow curse.
