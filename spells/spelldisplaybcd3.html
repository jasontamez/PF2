Rapid Repair - Spells Rapid Repair Source Ultimate Magic pg. 234 School transmutation; Level arcanist 5, cleric 5, occultist 4, oracle 5, sorcerer 5, warpriest 5, wizard 5 Casting Casting Time 1 standard action Components V, S, DF Effect Range touch Target construct touched Duration 1 round/level Saving Throw Fortitude negates (harmless); Spell Resistance yes (harmless) Description The targeted construct gains fast healing 5.
