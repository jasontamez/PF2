Sunstalker - Spells Sunstalker Source Adventurer's Guide pg. 17 School illusion (glamer); Level arcanist 3, bard 3, hunter 3, ranger 3, skald 3, sorcerer 3, wizard 3 Casting Casting Time 1 standard action Components V, S, M/DF Effect Range touch Target creature touched Duration 1 round/level Saving Throw Will negates (harmless); Spell Resistance yes Description A touched creature becomes invisible (as per invisibility) as long as it is in direct sunlight (or bright light shed by a daylight spell). If an action the target creature takes causes the invisible creature to become visible but does not end the sunstalker spell's duration, the creature can use this spell's magic to become invisible again as a standard action as long as it stands in an area of sunlight (as defined above). The creature gains only concealment (20% miss chance) in normal light.
