Psychic Sensitivity - Feats Psychic Sensitivity Source Occult Adventures pg. 138 You unlock the secrets of the occult world. Benefit: You gain access to occult skill unlocks (see page 194) for any skills in which you have ranks. If you have no ranks in the appropriate skill, you can't use the occult skill unlock, even if that skill can be used untrained. Normal: You must have the ability to cast psychic spells in order to use occult skill unlocks.
