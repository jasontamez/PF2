Basic Phytokinesis - Wild Talents - Kineticist Basic Phytokinesis Source Ultimate Wilderness pg. 58, Occult Origins pg. 8 Element wood; Type utility (Sp); Level 1; Burn 0 You can prune and otherwise tend plants within 30 feet without using gardening tools. You can search wooded areas and other plant-heavy areas from a distance as if using the sift cantrip. By concentrating, you can detect plants within 120 feet as if using detect animals or plants.
