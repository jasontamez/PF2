Feats Armor Mastery Feats Description Source Armor Master's Handbook pg. 16 Armor mastery feats are a new type of feat that require armor training as a prerequisite. They count as combat feats for all purposes, including which classes can select them as bonus feats. You gain the benefits of an armor mastery feat only while wearing armor with which you are proficient and only while wearing a type of armor that matches the feat's armor proficiency feat prerequisite, if any. Armor mastery feats without armor proficiency prerequisites can be used while wearing any suit of armor. Characters who lack the armor training class feature can access armor mastery feats by taking the Armor Focus feat, presented below.
