Weapon Style Mastery - Feats Weapon Style Mastery (Style, Weapon Mastery) Source Weapon Master's Handbook pg. 23 You can combine multiple fighting styles together. Prerequisites: Any two style feats from different styles, base attack bonus +6, weapon training class feature with a melee weapon. Benefit: Choose one weapon style (a style feat that lists Weapon Focus as a prerequisite) that you have. You can have the chosen style and a second style active at once. Starting a stance provided by a style feat is still a swift action, but you can assume both the chosen weapon style's stance and another style's stance simultaneously using this action. This ability doesn't stack with other abilities that allow you to have multiple styles active at the same time. Normal: You can have only one style active at once.
