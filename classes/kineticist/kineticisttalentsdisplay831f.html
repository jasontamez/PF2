Gravity Master - Wild Talents - Kineticist Gravity Master Source Occult Origins pg. 7 Element void; Type utility (Sp); Level 9; Burn 0 Prerequisite gravity control, greater gravity control You can create areas of altered gravity. This functions as reverse gravity, but you can choose for the gravity in the area to pull toward any direction (not just up). If you use this wild talent, any previous use of this wild talent expires immediately.
