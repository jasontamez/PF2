Blade Whirlwind - Wild Talents - Kineticist Blade Whirlwind Source Psychic Anthology pg. 20 Element universal; Type form infusion; Level 5; Burn 3 Prerequisite kinetic blade Associated Blasts any Saving Throw none You create a kinetic blade and swing it in an arc around you, making a single attack against all foes within reach as if with kinetic blade. If the result is a critical threat, roll to confirm the critical hit against only the first target you hit. The blade vanishes instantly after the whirlwind.
