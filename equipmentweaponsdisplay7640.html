Hooked axe - Weapons - Equipment Hooked axe Source Ultimate Equipment pg. 20, Ultimate Combat pg. 144 Statistics Cost 20 gp Weight 7 lbs. Damage 1d6 (small), 1d8 (medium); Critical x3; Range —; Type S; Special disarm, performance, trip Category One-Handed; Proficiency Exotic Weapon Groups Axes Description The hooked end of this axe's blade can be used to disarm foes or pull them off balance. A hooked axe can be used as a martial weapon (in which case it functions like a battleaxe).
