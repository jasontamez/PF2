Remove Blindness/Deafness - Spells Remove Blindness/Deafness Source PRPG Core Rulebook pg. 332 School conjuration (healing); Level alchemist 3, cleric 3, investigator 3, mesmerist 3, oracle 3, paladin 3, shaman 3, spiritualist 3, warpriest 3, witch 3 Casting Casting Time 1 standard action Components V, S Effect Range touch Target creature touched Duration instantaneous Saving Throw Fortitude negates (harmless); Spell Resistance yes (harmless) Description Remove blindness/deafness cures blindness or deafness (your choice), whether the effect is normal or magical in nature. The spell does not restore ears or eyes that have been lost, but it repairs them if they are damaged. Remove blindness/deafness counters and dispels blindness/deafness.
