Wand Key Ring - Wondrous Items - Magic Equipment Wand Key Ring Source Advanced Class Guide pg. 237 Aura faint universal; CL 5th Slot none; Price 3,000 gp; Weight 8 lbs. Description Each of these rune-carved brass rings is keyed to a specific spell at creation. When using the Use Magic Device skill to activate a scroll or spell trigger item based on that spell, the bearer gains a +10 insight bonus on the skill check. Construction Requirements Craft Wondrous Item, guidance, creator must have 10 ranks in the Use Magic Device skill; Cost 1,500 gp
