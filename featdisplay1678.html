Dual Path - Feats Dual Path Source Mythic Adventures pg. 63 You follow two mythic paths. Prerequisites: 1st mythic tier. Benefit: Select a mythic path other than the path you selected at your moment of ascension. You gain that path's 1st-tier ability (either archmage arcana, champion's strike, divine surge, guardian's call, marshal's order, or trickster attack). Each time you gain a path ability, you can select that path ability from either path's list or the list of universal path abilities.
