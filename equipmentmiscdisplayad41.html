Miscellaneous - Equipment Thieves' tools Source Ultimate Equipment pg. 77, PRPG Core Rulebook pg. 158 Price 30 gp (common), 100 gp (masterwork); Weight 1 lb. (common), 2 lbs. (masterwork) Description This kit contains lockpicks and other tools you need to use the Disable Device skill. Without these tools, you must use improvised tools, and you take a -2 circumstance penalty on Disable Device checks. Masterwork Thieves' Tools: This kit contains extra tools and tools of better make, which grant a +2 circumstance bonus on Disable Device checks.
