Handle Animals Source Pathfinder Unchained pg. 61 This use of Nature functions as the Handle Animal core skill, but uses Intelligence instead of Charisma. Action: Varies. Handling an animal is a move action, while "pushing" an animal is a full-round action. Try Again: Yes, except for rearing an animal.
