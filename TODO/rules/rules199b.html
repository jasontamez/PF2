An entry marked with this has additional sections within it. Post Game Source GameMastery Guide pg. 26 Once the game session is done, there are still several tasks that need to be taken care of. First, award the players any experience earned during play, and allow them to divide up any loot earned. Having your players come up with an equitable wealth distribution system they can all agree on is vital to the long-term health of the campaign, so help them come up with a system that works for all involved. Last but not least, have the players help you clean up the gaming area. Once the players have gone home, it may be a good idea to do some basic record-keeping, such as writing down what happened that session in case you need to refresh your memory later, or plot ideas to revisit in future sessions.
