Prerequisites Source Pathfinder Unchained pg. 67 If a feat, prestige class, or other ability requires a certain number of ranks in a skill, directly convert that core skill to the consolidated skill. Use the same number of ranks. For example, the Mounted Combat feat would require Acrobatics 1 rank instead of Ride 1 rank.
