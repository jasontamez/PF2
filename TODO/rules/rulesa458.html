An entry marked with this has additional sections within it. Definitions of Terms Source PRPG Core Rulebook pg. 374 Here are definitions of some terms used in this section. Core Class: One of the standard eleven classes found in Chapter 3. Caster Level: Generally equal to the number of class levels (see below) in a spellcasting class. Some prestige classes add caster levels to an existing class. Character Level: The total level of the character, which is the sum of all class levels held by that character. Class Level: The level of a character in a particular class. For a character with levels in only one class, class level and character level are the same.
