Magnetism - Wild Talents - Kineticist Magnetism Source Elemental Master's Handbook pg. 22 Element air, earth; Type utility (Sp); Level 3; Burn 0 Prerequisite magnetic infusion Saving Throw none; SR yes You can push and pull metal objects using magnetism. This functions as telekinetic maneuvers but you can make only bull rush and drag combat maneuvers, and can do so only against metal objects or creatures. If you use this ability on an attended metal object, you must exceed the creature's CMD as normal, and if it's holding the object, it must drop it or this ability pushes or pulls the creature.
