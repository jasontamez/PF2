Fervor juice - Miscellaneous - Equipment Fervor juice Source Monster Codex pg. 117 Price 50 gp; Weight — Category Alchemical Tools Description This foul-tasting alchemical concoction grants those that imbibe it ferocity for a short period of time. This functions as the universal monster rule of the same name, and lasts for 1 hour from the time you ingest the solution. Crafting this item requires a successful DC 20 Craft (alchemy) check.
