Bullet, screaming - Weapons - Equipment Bullet, screaming Source Heroes of Golarion pg. 18 Statistics Cost 20 gp Weight — Category Ammunition; Proficiency Exotic Description This ammunition makes a piercing scream when fired. If fired as part of an Intimidate check, it imparts a +5 bonus on the check. If used with the startling shot deed, the target is flat-footed for an additional round.
