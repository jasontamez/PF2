Miscellaneous - Equipment Artisan's tools Source Ultimate Equipment pg. 77, PRPG Core Rulebook pg. 158 Price 5 gp (common), 55 gp (masterwork); Weight 5 lbs. (common), 5 lbs. (masterwork) Description These special tools include the items needed to pursue any single craft. Without them, you have to use improvised tools (-2 penalty on Craft checks), if you can do the job at all.
