Dragoncrafting kit - Miscellaneous - Equipment Dragoncrafting kit Source Dragonslayer's Handbook pg. 26 Price 40 gp; Weight 4 lbs. Category Tools Description This kit contains all the butchers' tools, preservatives, and desiccants necessary to turn a dragon corpse into usable items. It provides the user with a +2 circumstance bonus on all Heal and Craft skill checks made to harvest dragoncraft materials and create dragoncraft items (see page 18).
