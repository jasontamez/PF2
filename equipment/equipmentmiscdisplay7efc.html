Fangfile - Miscellaneous - Equipment Fangfile Source Goblins of Golarion pg. 12 Price 3 gp; Weight 2 lbs. Category Adventuring Gear Description This narrow metal file is used by goblins to sharpen their teeth. The file ends in a pointed spike in case tooth-sharpening is interrupted by combat-a fangfile can function as a punching dagger when used in combat.
