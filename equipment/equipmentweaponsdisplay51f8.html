Damage 1d4 (small), 1d6 (medium); Critical x2; Range —; Type B; Special disarm, monk Category Light; Proficiency Martial Weapon Groups Monk Description This batonlike weapon is fitted with a metal hook that can be used to block and pin an opponent's weapon.
